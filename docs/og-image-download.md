# Как скачать `og.png`

Файл `og.png` находится в корне репозитория. Если репозиторий у вас уже на компьютере, просто откройте файл как обычную картинку:

- **Windows:** `start og.png` (или двойной клик в Проводнике)
- **macOS:** `open og.png`
- **Linux:** `xdg-open og.png`

## Вариант 1: Скачать через веб-интерфейс репозитория
1. Откройте репозиторий в веб-интерфейсе (GitHub/GitLab/и т.д.).
2. Перейдите в файл `og.png` в корне проекта.
3. Если видите сообщение **«Бинарный файл не отображается»**, это нормально — нажмите **Download** (или **Raw**) и сохраните файл.

## Вариант 2: Если вы работаете в облачной среде (Codespaces/Devcontainer)
1. Откройте файловый браузер среды.
2. Перейдите в корень репозитория.
3. Выберите `og.png` и нажмите **Download**.

## Вариант 3: Если бинарные файлы блокируются (текстовый вариант в репозитории)
В репозитории лежит **текстовый** файл `scripts/og-image/og.png.b64`. Его можно декодировать в PNG одной командой:

```bash
node scripts/og-image/decode-og-image.mjs
```

После этого в корне репозитория появится `og.png`.

## Вариант 4: Через SSH/SCP (если у вас есть доступ к удалённой машине)
Укажите реальный хост и путь к репозиторию на удалённой машине:

```bash
scp user@<remote-host>:/path/to/dreddi-mvp/og.png .
```

> Важно: путь `/workspace/dreddi-mvp` существует только внутри контейнера. На вашей локальной Windows-машине его нет, поэтому `cd /workspace/dreddi-mvp` не сработает.
